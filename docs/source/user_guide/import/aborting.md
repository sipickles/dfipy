# Aborting an Import

:::{note}
A finished import cannot be aborted.
:::

:::{warning}
An abort request occurs immediately. Any data sucessfully ingested prior to issuing an abort command is not rolled back.
:::

## Example

::::{tab-set}

:::{tab-item} dfipy
:sync: dfipy

```python
from dfi import Client

dfi = Client(<token>, <url>)

status = "aborted"
dfi.ingest.update_batch_status(<import_batch_id>, status)
```
